Comandos git:

git init

O comando "git init" cria um repositório git na pasta que você
quer enviar para o Github. Ou seja, a pasta será um projeto gerenciado pelo git.

git add .

O comando "git add ." vai enviar os arquivos da pasta para uma área
temporária chamada "stage".

git status

O comando "git status" mostra as modificações feitas para ser "comitadas", ou seja,
para serem salvas
